Cyber War Scenario - 2017

This was a Capstone project intended to be a training application for students in the Cyber Security course at Marshall University. Scenarios are typical real world issues written as Bash shell Scripts. By utilizing a Linux distribution on a Virtual Machine, the students can create a snapshot of their workspace before starting on any scenario and have the option to roll back their OS in the event of critical error. Saving a user file on the host machine after each scenario allows the student to be graded on their work without having to redo any scenario.

Features include:

  • Admin Login to Grade Students and Add Custom Scenarios
  • Capable of Adding Many Self-Attacking Scenarios
  • List of example Scenarios
  • Currently Being Expanded for Network and Web Scenarios
